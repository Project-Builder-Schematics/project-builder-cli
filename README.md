# project-builder-cli

[![CI](https://github.com/Project-Builder-Schematics/project-builder-cli/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/Project-Builder-Schematics/project-builder-cli/actions/workflows/ci.yml)

Modern schematics runner built in Go. Open source interface for the Project Builder ecosystem — generates and orchestrates project scaffolds via composable AI skills, with Angular schematics as the first execution backend.

## Status

Pre-`v1.0` — repository is being bootstrapped. No releases yet. The legacy TypeScript implementation lives at [`@pbuilder/cli`](https://www.npmjs.com/package/@pbuilder/cli) (v1.9.4) and is being rewritten in Go for distribution as platform-specific binaries via npm wrappers.

## Install

Pending `v1.0`. Once cut, the CLI will install via:

```sh
npm i -g @pbuilder/cli
```

To track or contribute to the rewrite in the meantime, see [`CONTRIBUTING.md`](./CONTRIBUTING.md).
